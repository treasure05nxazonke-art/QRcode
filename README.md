âš ï¸ Challenges Faced

I faced a few challenges while working on this project:

Padding:
I struggled to balance padding so that the content looked neat without making the card too large or too small.

Min-width and Max-width:
Understanding how min-width, max-width, and fixed widths interact took some trial and error. Sometimes the container did not resize the way I expected.

Max-height:
I had issues controlling the height of the container, especially when combining max-height with flexible content like text and images.

I solved these problems by testing different values, using browser developer tools, and learning how Flexbox affects layout behavior.
